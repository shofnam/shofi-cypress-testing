# Cypress Automation Testing Project

# Project Overview
Project ini merupakan implementasi automation testing menggunakan Cypress yang mencakup:

- API Testing
- UI Testing
- Network Interception Testing
- Page Object Model (POM)

Pengujian dilakukan pada:
- Public API (Categories API)
- Website OrangeHRM (Login Feature)

Tujuan dari project ini adalah untuk memvalidasi fungsionalitas sistem melalui berbagai skenario pengujian, baik positif maupun negatif.
